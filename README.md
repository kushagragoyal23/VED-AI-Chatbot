# 🤖 VED - Rule-Based AI Chatbot

VED is a rule-based AI chatbot built using Python that responds to predefined user queries through conditional logic.

This project was developed as part of my Artificial Intelligence Internship at Decode Labs to strengthen my understanding of Python fundamentals, chatbot logic, and user interaction.

---

## ✨ Features

- 👋 Greets users with predefined responses
- 💬 Answers basic questions
- 🔁 Supports continuous conversation using a loop
- 🚪 Allows users to exit anytime by typing `bye`
- ❌ Handles unknown inputs gracefully with a default response

---

## 🛠️ Technologies Used

- Python
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`while`)
- User Input Handling
- Rule-Based Chatbot Logic

---

## 🚀 How to Run

1. Install Python.
2. Download the `ved.py` file.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python ved.py
```
